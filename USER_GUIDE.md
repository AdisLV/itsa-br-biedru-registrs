# USER GUIDE – Lietotāja pamācība

## 1. Kas šī ir par sistēmu?
Šī ir vienkārša tīmekļa lietotne, kas parāda arodbiedrības biedru sarakstu no Supabase datubāzes. Sistēma ļauj tikai skatīties datus – neko nevar mainīt, dzēst vai pievienot.


## 2. Kā atvērt lietotni?
1. Atver tīmekļa pārlūkprogrammu (Chrome, Edge, Firefox).
2. Adreses joslā ievadi projekta GitHub Pages adresi:  
   `https://adislv.github.io/itsa-br-biedru-registrs/`
3. Kad lapa ielādējas, automātiski tiek parādīts biedru saraksts.


## 3. Ko lietotājs redz?
Lietotnē būs redzams saraksts tabulas formā:

| ID | Vārds | Statuss |
|----|-------|---------|
| 1 | Jānis Bērziņš | Aktīvs |
| 2 | Anna Liepa | Neaktīvs |

- **ID** — unikāls biedra numurs
- **Vārds** — biedra vārds un uzvārds
- **Statuss** — biedra dalības statuss


## 4. Iespējamās darbības lietotājam
| Darbība | Pieejama šajā versijā |
|---------|------------------------|
| Skatīt biedru sarakstu | Jā |
| Atvērt lietotni ar pārlūkprogrammu | Jā |
| Pievienot jaunus biedrus | Nē |
| Labot vai dzēst biedrus | Nē |
| Meklēt vai filtrēt biedrus | Nē |
| Saglabāt sarakstu PDF/Excel | Nē |


## 5. Biežāk uzdotie jautājumi

** Ja lapā nerāda datus?**  
 Pārbaudi, vai ir interneta savienojums  
 Pārlādē lapu  
 Ja Supabase atslēga izņemta — dati nerādīsies

** Vai drīkst ievadīt jaunus biedrus?**  
 Nē, šī ir tikai skatīšanās versija

** Vai dati tiek saglabāti datorā?**  
 Nē, dati paliek Supabase datubāzē


## 6. Secinājums
Šī sistēma paredzēta, lai ērti apskatītu arodbiedrības biedru sarakstu no datubāzes. Tā ir vienkārša, droša un viegli izmantojama mācību vai demonstrācijas nolūkos.
